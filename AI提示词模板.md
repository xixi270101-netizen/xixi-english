# AI 提示词模板

复制粘贴用。准备两套 prompt：一套周批量翻译用，一套单条精调用。

---

## 模板1：周末批量翻译（推荐）

```
你是一个英语教练。我要做一个英语口播账号，每天用1-3分钟英语聊一个情感话题。我的英语水平在A2-B1。

以下是7段中文，每一段是我对一个话题的随口表达。请帮我：
1. 每段翻译成适合A2-B1水平的英语口语
2. 用最简单的词汇和短句
3. 每句话不超过15个词
4. 保留我的语气和情绪，不要变成书面语
5. 在每段末尾用（括号）列出3个我可能不会读的词，给出音标
6. 在每句英文下面用【中文意思】标注，方便我理解

请按以下格式输出：
---
Day1 话题：{话题名}
英文稿：
...
笔记：（列词+音标）
---

以下是我的7段中文：
{粘贴7段中文}
```

## 模板2：单条精调（当某条需要深度修改时用）

```
你是一个英语教练。我要把下面这段中文想法，用适合A2-B1水平的英语来表达。

要求：
1. 用最简单常见的词，能用人称代词就不用复杂名词
2. 每句话不超过15个词
3. 用现在时和过去时为主，不要用完成时
4. 保留口语感——要有停顿、语气词的感觉
5. 如果某句英文表达不够自然，给我2个替换说法

我的中文：
{粘贴中文}
```

---

## 模板3：口语化改写（AI翻译太书面时用）

```
下面这段英文太书面了，请帮我改口口语化版本：

要求：
1. 用口语常用词
2. 加入自然的停顿词（like, you know, I mean, honestly, actually）
3. 可以用不完整句
4. 每句不超过12个词
5. 读起来像在跟朋友说话

原文：
{粘贴太书面的英文}
```

## 模板4：检查你是否能读出来

```
我准备录一条英语口播。这是我的最终稿。请帮我：
1. 标出普通A2水平的人可能不会读的词，给音标
2. 标注哪些句子太长可能需要断句（用 / 标出断句位置）
3. 如果有更好的简单词可以替换，请告诉我

我的稿子：
{粘贴你的英语终稿}
```

1. 哪条数据最好？可能的原因是什么？
2. 哪条数据最差？为什么没人看？
3. 下周话题方向建议

本周数据：

## 使用建议

- **翻译**：周末批量用模板1，一次搞定7条
- **改稿**：AI给英文后，你还是要自己改成你能说出口的话——这一步不能省，这是你学英语的核心环节
- **口播前**：不确定怎么读时用模板4帮你过一遍
- **复盘**：每周日用模板5
