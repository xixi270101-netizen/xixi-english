# 每日执行清单

每天打开这一页，按顺序执行，做完打 ✅

---

## 每天任务（30分钟）

### 执行前检查
- [ ] 手机存储空间够录视频吗？
- [ ] 今天的话题稿昨天已经准备好了吗？

---

### Step 1：拿稿（1分钟）
- [ ] 打开今天的话题英文稿

### Step 2：改成你的话 ⭐（8分钟）
- [ ] 读一遍AI翻译稿
- [ ] 圈出不认识/不会读的词 → 换简单词
- [ ] 长句拆短（每句 ≤ 15词）
- [ ] 加上你的语气词和停顿
- [ ] 大声读一遍改好的版本
- [ ] 把终稿写在最下面

### Step 3：练2遍（5分钟）
- [ ] 第1遍：看稿读顺
- [ ] 第2遍：尽量少看稿

### Step 4：录制（10分钟）
- [ ] 光线OK（面朝窗户/灯）
- [ ] 手机横屏 or 竖屏固定好
- [ ] 录第1遍（不用完美）
- [ ] 不满意再录1-2遍
- [ ] 选最好的一条

### Step 5：剪辑+发布（5分钟）
- [ ] 剪映自动字幕
- [ ] 检查字幕错误
- [ ] 写标题（中英混合，有情绪）
- [ ] 小红书发布
- [ ] 抖音发布

### 执行后记录
- [ ] 去 [进度追踪.md](进度追踪.md) 打勾
