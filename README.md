# 溪溪的英语账号训练计划

## 账号定位

> 一个中国审计女孩，用不完美但真实的英语，聊那些中文里说不出口的成长话题。

**三个标签**：在职学英语 | 情绪成长 | 真实不装

---

## 核心工作流（总览）

```
周末批量备料（1h） → 工作日每天执行（30min） → AI做翻译层
```

| 阶段 | 做什么 | 何时 |
|---|---|---|
| 周末备料 | 选7个话题 + 语音转文字写中文 + AI翻译 + 存素材库 | 周六或周日 |
| 每天执行 | 拿稿→改成自己的话→练2遍→录→剪→发 | 每天晚上 |
| 周复盘 | 看数据 + 挑出最好的一条 + 定下周话题方向 | 周日 |

---

## 每天具体步骤（30分钟）

### Step 1：拿AI翻译稿（1分钟）
打开今天那条话题的AI翻译稿。

### Step 2：改成你的话（8分钟）⭐关键步骤
- 读一遍AI翻译，圈出你不会读、不会用的词
- 换成你会的简单词
- 长句拆成短句（每句不超过15个词）
- 加上你的语气词和自然停顿
- **这个修改过程本身就是在学英语**

### Step 3：练2遍（5分钟）
- 出声读你改好的稿子
- 标记卡顿的地方，再改顺
- 第二遍尝试减少看稿次数

### Step 4：录制（10分钟）
- 手机直接录，1-3分钟
- 不看稿（可以瞟一眼关键词）
- 说错了继续说，不要停
- 可以录2-3遍，选最好的一遍

### Step 5：剪辑+发布（5分钟）
- 剪映自动生成字幕
- 加一个标题（中英混合）
- 小红书 + 抖音同步发布

## 周末批量备料（1小时）

### 准备材料
从 [话题库.md](话题库.md) 中选下周7天的话题。

### 语音转文字写中文（35分钟）
- 打开手机语音备忘录
- 对着每个话题随口说2分钟中文
- 用飞书妙记/讯飞听见转成文字
- **不用修饰，口语化就是最好的**

### AI翻译（5分钟）
- 7段中文一次性发，用 [AI提示词模板.md](AI提示词模板.md) 里的 prompt
- 把返回的英文稿存进飞书/Notion/备忘录

### 存素材库（10分钟）
```
素材库结构：
├── Week1/
│   ├── Day1_话题.md （中+英对照）
│   ├── Day2_话题.md
│   └── ...
├── Week2/
└── ...
```

## 分阶段目标

### 第一阶段：练手期（6月-7月）
- **唯一目标**：形成肌肉记忆
- **核心指标**：
  - 今天录了吗？ YES/NO
  - AI改稿时间是否下降？（目标：< 3分钟）
  - 能不能不看稿说1分钟？
- **变现**：0。这个阶段不想钱。
- **发布**：建议发。发了就当存档。

### 第二阶段：稳定期（8月-9月）
- **目标**：找到你的"爆款话题类型"
- **核心指标**：
  - 点赞/收藏/评论数据
  - 什么话题数据好？什么没人看？
- **粉丝目标**：500-1000（够开蒲公英）
- **试探变现**：发一条"如果我想做一个英语情绪表达小组有人感兴趣吗"

### 第三阶段：变现期（10月-12月）
- **变现动作**：
  1. 英语情绪表达陪跑营（21天，299元/人，限30人）= 8970元
  2. 60个中英表达模板打包（39元 × 100份）= 3900元
  3. 1v1聊天陪练（99元/次 × 20次）= 1980元
  4. 蒲公英广告（200-500元/条 × 10条）= 3000元
- **合计**：约1.8万（保守），冲刺5万需要陪跑营扩量或出爆款
- **粉丝目标**：3000-5000

## 坚持法则

### 1. 不完美发布原则
录完觉得英语不够好、衣服不够好看、光线不好 → 发。完成比完美重要100倍。

### 2. 备胎机制
加班到10点 → 发一条30秒的"今日情绪碎片"。保持惯性 > 内容质量。

### 3. 只看过程，不看结果
前2个月不看粉丝数、不看播放量。只看：
- 今天录了吗？
- 能用英语说的话变多了吗？
- AI改稿时间缩短了吗？

### 4. 这不是号，是日记
做不起来也没关系，核心是练英语。越松弛，内容越好看。
